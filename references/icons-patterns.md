# Icons & Patterns (Curated for archviz 2D info viz)

## Recommended Icon Sets (Clean, Restrained SVG for Diagrams)
From 400-free-design-resources (Icons category):
- Lucide (https://lucide.dev) — 1,400+ beautiful open-source (Feather fork). MIT.
- Heroicons (https://heroicons.com) — Hand-crafted by Tailwind team. Perfect for UI/diagram nodes.
- Tabler Icons (https://tabler.io/icons) — 5,000+ MIT SVG. Flexible weights.
- Phosphor Icons (https://phosphoricons.com) — Multiple weights, open.
- Radix Icons (https://www.radix-ui.com/icons) — Crisp 15×15, accessible.
- css.gg (https://css.gg) — Pure CSS/SVG, 700+.
- Teenyicons (https://teenyicons.com) — Tiny minimal 15×15.
- Akar Icons (https://akaricons.com) — Perfectly rounded.

**Usage**: Prefer line/solid variants for flowcharts, architecture diagrams. Add to assets/ as needed. Avoid emoji or overly decorative.

## Patterns, Textures & SVG Generators (For Backgrounds in HTML Viz)
- Hero Patterns (https://heropatterns.com) — Repeatable SVG backgrounds. Free.
- Subtle Patterns (https://www.toptal.com/designers/subtlepatterns) — Tileable, subtle.
- Haikei (https://haikei.app) — Generative SVG (blobs, waves, backgrounds). Great for restrained viz.
- SVGBackgrounds (https://www.svgbackgrounds.com)
- Pattern Monster, Magic Pattern, BGJar for generators.

**Gotchas**:
- Overuse of patterns kills restraint — use sparingly, max 1 accent layer.
- Always test export (4x raster) for pattern crispness.
- Prefer SVG over images for self-contained HTML.
